# NFG companion IPAs (sideload / website download)

| File | App | Bundle ID |
|------|-----|-----------|
| `NFG-Crash.ipa` | NFG Crash | `com.nfg.crash` |
| `NFG-Hangman.ipa` | NFG Hangman | `com.nfg.hangman` |

Built on Mac with `scripts/build-all-ipas.sh` (Debug export for device testing).

**Install:** Settings → General → VPN & Device Management → trust developer, or use https://y666suf.com/sideload

**Server:** Node serves these from `releases/ipa/` when present, else `~/Downloads/NFG-*.ipa` on the PC.
