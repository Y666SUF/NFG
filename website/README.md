# Host privacy policy on y666suf.com

1. Upload **`privacy.html`** to your web host so it is available at:
   ```
   https://y666suf.com/privacy
   ```
   (or `https://y666suf.com/privacy.html` — configure redirects so `/privacy` works.)

2. In **App Store Connect**, set Privacy Policy URL to:
   ```
   https://y666suf.com/privacy
   ```

3. Edit **`privacy@y666suf.com`** in:
   - `website/privacy.html`
   - `ios/NFGCrash/Legal/PrivacyPolicyContent.swift`  
   Replace with your real contact email before going live.

The same text is shown in the app under **Wallet → Legal & compliance → Privacy Policy**.
