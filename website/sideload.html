Sideload NFG Crash on iPhone

This guide is for test installs before App Store release. Download the latest IPA and install using Sideloadly (Windows/Mac) or AltStore.

Download NFG Crash .ipa

Method 1: Sideloadly (recommended on Windows)

  • Install iTunes + iCloud from Apple website (not Microsoft Store).
  • Install Sideloadly from sideloadly.io.
  • Connect iPhone to PC, unlock device, tap Trust if prompted.
  • Open Sideloadly, choose your device, drag in the downloaded `.ipa`.
  • Sign with your Apple ID and start sideload.
  • On iPhone: Settings -> General -> VPN & Device Management -> Trust your developer profile.
  • On iOS 16+: enable Developer Mode in Settings -> Privacy & Security.
