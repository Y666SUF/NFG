# NFG companion IPAs

| File | App | Bundle ID |
|------|-----|-----------|
| `NFG-Crash.ipa` | NFG Crash | `com.nfg.crash` |
| `NFG-Hangman.ipa` | NFG Hangman | `com.nfg.hangman` |

Built on Mac (`scripts/build-all-ipas.sh`, Debug export for device testing).

**Install:** https://y666suf.com/sideload — or trust developer in Settings after installing.

**Server:** Node serves from this folder first (`server/ipa-downloads.js`), then `%USERPROFILE%\Downloads\` on Windows.
